## 🌟 Как это работает

**Kindle Vocabulary to Anki Sync Hub** — это мощный инструмент, созданный для того, чтобы связать чтение на Kindle и изучение языков с помощью Anki.

Базово, приложение предназначено для **создания карточек с переводом и определениями** из слов, которые вы искали во время чтения на Kindle.

1. **Загрузите ваш `vocab.db`**: Подключите Kindle и перетащите файл базы данных словаря в приложение. База обрабатывается на 100% локально в вашем браузере.
2. **Перевод и значения**: Мы используем бесплатные сервисы без ИИ (Google Translate, MyMemory и Free Dictionary API) для получения перевода и значений слов.
3. **Просмотр и синхронизация**: Проверьте сгенерированные карточки, выберите нужные языки и отправьте их напрямую в Anki!

## ✨ Ключевые фишки

### 🃏 Создание карточек
Основная функция — автоматическое создание качественных флеш-карточек с переводами и контекстом (предложениями из книг), в которых вы искали это слово.

### 🔄 Умное сравнение с Anki
Никаких дубликатов! В приложении есть дополнительная возможность сравнить базу данных Kindle с вашей колодой в Anki. Это позволяет не дублировать и пропускать те слова, которые вы уже добавили и изучаете.

### ✏️ Редактирование и экспорт в стиле Kindle
Вы можете не только экспортировать слова в Anki, но и **редактировать базу данных и экспортировать её обратно в формате Kindle**. Это позволяет легко чистить и управлять оригинальным словарем (`vocab.db`) прямо на вашем устройстве.

### 🔒 100% Приватность
Ваша база данных обрабатывается на стороне клиента с помощью WebAssembly (`sql.js`). Ваш файл `vocab.db` никуда не загружается и остается только на вашем компьютере.

## 🚀 Запуск локально

**Требования**: [Node.js](https://nodejs.org/)

1. Установите зависимости:
   ```bash
   npm install
   ```
2. Запустите локальный сервер:
   ```bash
   npm run dev
   ```
3. Откройте браузер и перейдите на `http://localhost:3000`, чтобы начать синхронизацию!

## ☁️ Статический хостинг (Serverless)

Это приложение может работать **полностью на стороне клиента** в вашем браузере без использования Node-бэкенда.

Чтобы задеплоить на статический хостинг (GitHub Pages, Vercel, Netlify, Cloudflare Pages и т.д.):

1. Соберите продакшн-билд:
   ```bash
   npm run build
   ```
2. Загрузите содержимое папки `dist/` на ваш хостинг-провайдер.
3. Фронтенд будет автоматически выполнять прямые запросы для переводов и определений на стороне клиента.
