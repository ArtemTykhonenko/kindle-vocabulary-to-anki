## 🌟 Як це працює

**Kindle Vocabulary to Anki Sync Hub** — це потужний інструмент, створений для того, щоб поєднати читання на Kindle та вивчення мов за допомогою Anki.

Базово, програма призначена для **створення карток з перекладом та визначеннями** зі слів, які ви шукали під час читання на Kindle.

1. **Завантажте ваш `vocab.db`**: Підключіть Kindle і перетягніть файл бази даних словника в додаток. База обробляється на 100% локально у вашому браузері.
2. **Переклад та значення**: Ми використовуємо безкоштовні сервіси без ШІ (Google Translate, MyMemory та Free Dictionary API) для отримання перекладу та значень слів.
3. **Перегляд та синхронізація**: Перевірте згенеровані картки, виберіть потрібні мови та відправте їх безпосередньо в Anki!

## ✨ Ключові фішки

### 🃏 Створення карток
Основна мета — автоматичне створення якісних флеш-карток з перекладом та контекстом (реченнями з книг), в яких ви шукали це слово.

### 🔄 Розумне порівняння з Anki
Жодних дублікатів! У додатку є додаткова можливість порівняти базу даних Kindle з вашою колодою в Anki. Це дозволяє не дублювати та пропускати ті слова, які ви вже додали і вивчаєте.

### ✏️ Редагування та експорт у стилі Kindle
Ви можете не тільки експортувати слова в Anki, але й **редагувати базу даних і експортувати її назад у форматі Kindle**. Це дозволяє легко чистити та керувати оригінальним словником (`vocab.db`) прямо на вашому пристрої.

### 🔒 100% Приватність
Ваша база даних обробляється на стороні клієнта за допомогою WebAssembly (`sql.js`). Ваш файл `vocab.db` нікуди не завантажується і залишається тільки на вашому комп'ютері.

## 🚀 Запуск локально

**Вимоги**: [Node.js](https://nodejs.org/)

1. Встановіть залежності:
   ```bash
   npm install
   ```
2. Запустіть локальний сервер:
   ```bash
   npm run dev
   ```
3. Відкрийте браузер і перейдіть на `http://localhost:3000`, щоб розпочати синхронізацію!

## ☁️ Статичний хостинг (Serverless)

Цей додаток може працювати **повністю на стороні клієнта** у вашому браузері без використання Node-бекенду.

Щоб задеплоїти на статичний хостинг (GitHub Pages, Vercel, Netlify, Cloudflare Pages тощо):

1. Зберіть продакшн-білд:
   ```bash
   npm run build
   ```
2. Завантажте вміст папки `dist/` на ваш хостинг-провайдер.
3. Фронтенд автоматично виконуватиме прямі запити для перекладів та визначень на стороні клієнта.
